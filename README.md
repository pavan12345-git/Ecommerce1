# E-commerce Products API

## Endpoints

### Get All Products
`GET /api/products`

**Parameters:**
- `page` (default: 1)
- `per_page` (default: 10)

### Get Single Product
`GET /api/products/<id>`

## Setup
1. Clone repository
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`
